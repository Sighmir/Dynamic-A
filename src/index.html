Controls:

Enter - Run Astar

Space - Clean path

Click - Add/remove wall

Shift + Space - Clean path and walls

Shift + Click - Add/remove start

Ctrl + Click - Add/remove end

Alt + Click - Add/remove connected cells

Alt + Shift + Click - Add/remove shared start

Alt + Ctrl + Click - Add/remove shared end

Alt + Enter - Run shared Astar

T - Toggle debug text
